<h2 align="center">Algorithm</h2>

```
- Create pointers -> left(l) and right(r)
- Create middle pointer(m) -> l+(r-l)/2
- While (array size > m)
  - Create middle number(mn) -> array[m]
  - Verifications
    - if (mn == target) return m
    - if (mn > target) r = m - 1
    - if (mn < target) l = m + 1
    - if (l > r) "Target not found"
  - m = l+(r-l)/2
- return -1 (if search don't start)
```
